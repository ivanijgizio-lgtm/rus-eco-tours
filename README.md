# RUSECOTOURS - Экотуры и треки по России

## Описание
**EcoRussTour** — агрегатор внутреннего туризма РФ: экотреки, агротуры, локальные гиды. Аналог Booking для регионов (Алтай, Байкал, Калмыкия). Рост 20-30% в 2026: экотуризм +25%, агротуры +30%.[web:35][web:36]

**Слоган:** "Открой Россию: природа, гиды, без посредников".

## Функции MVP
- **Неоморфный дизайн** (neumorphism): градиенты #27ae60, blur, тени.
- **Адаптивный** (mobile/tablet/desktop).
- **Поиск:** Input + фильтры (экотреки/гастро/агро).
- **Карточки туров:** Цены, регионы, длительность (как Tutu.ru).[web:42]
- **Мотивация:** "20-30% рост туризма!", иконки (гора, вода, трактор).
- **Данные:** Топ-направления 2026 (Краснодар 48%, Калмыкия +50%).[web:39]

## Технологии
- HTML5/CSS3/JS чистый.
- Font Awesome (иконки), Google Fonts (Inter).
- Градиенты/Backdrop-filter для eco-стиля.
